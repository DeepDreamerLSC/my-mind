# 项目总览

## 项目目标

构建面向教育场景的智能体服务：以单一聊天入口承载学生和老师的任务，通过后端能力发现、模型网关、文件上传下载、PDF 题目入库、OCR/版面理解、题目裁切和智能排版能力，支持后续错题、作文批改、学情分析、考情分析和组卷等教育任务。

## 当前阶段

当前重点是 `PDF题目入库与智能排版质量门禁阶段`：

- PDF 题目裁切和题目边界识别持续优化。
- Qwen/Qwen36、Apple Vision OCR、版面重排等后端能力在真实样例上做质量优先调参。
- 文件上传下载、OSS、本地文件存储、OpenAPI 和前端接口正在对齐。
- 生产/测试环境通过 Docker Compose、Cloudflare Tunnel 和环境模板进行分层管理。

## 成功标准

- PDF 练习题可以稳定转成可复核的题目结构和裁切资产。
- 质量门禁能阻止错切、漏题、跨页吞题和答案解析误并入题干。
- 前端能以低噪声方式完成文件上传、对话、题目预览和继续追问。
- 测试环境和生产环境配置可复现、可回滚、不会依赖本地临时状态。
- 关键设计、测试和运行证据能回写到本地项目管理，不只散落在代码仓库 commit 中。
