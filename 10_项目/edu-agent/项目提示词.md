# 项目提示词

## 默认协作提示词

请把当前任务放在 `edu-agent` 项目上下文中处理。

工作前先区分：

1. 这是 PDF 题目入库 / 裁切 / OCR / 版面重排问题，还是前端对接、文件存储、OpenAPI、部署配置或项目管理问题。
2. 证据来自代码、设计稿、测试、运行 artifact、日志还是用户反馈。
3. 当前操作是本地开发、测试环境、生产副本还是知识库项目管理。

执行原则：

- 质量大于速度，尤其是 PDF 题目裁切和结构化入库。
- 不把运行日志、大体积 artifact、密钥或生产配置写入知识库正文。
- 对外接口和前端对接必须保留最小可验证证据。
- 重要结论先写成项目候选，确认后再进入项目进展、决策或风险。
