# 变更证据

用于保存项目进展的证据索引，包括 git commit、重要工作区变更、运行记录和自动化报告。

## 证据规则

- commit 是证据，不等于项目进展结论。
- 运行记录是过程证据，不等于长期知识。
- 定时巡检可以追加候选证据，但不自动生成 commit。
