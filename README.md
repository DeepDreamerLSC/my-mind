# 我的心智

我的心智是一个个人数据资产系统，目标是把资料、人工智能对话、行业信号、项目记录、原子笔记、主动回忆、洞察和复盘组织成一个可长期运行的数据飞轮。

## 当前阶段

- 阶段一：最小结构
- 阶段二：处理当前设计讨论

## 核心文档

- [个人数据资产飞轮设计稿](design/个人数据资产飞轮设计稿.md)

## 可用技能

- `inbox-capture`：把视频链接、网页链接或粗略片段写入 `00_收件箱/`。第一版会收集标题、作者或频道、发布时间、时长、平台、简介摘录、标签、分类、缩略图和字幕可用性；YouTube 链接会尝试解析字幕轨道的来源、语言、格式、片段数和估算词数，也会尝试发现官方转录来源并提取章节目录；解析失败也会保留链接和失败原因。

## 第一版目录

- `00_收件箱/`：未处理内容入口
- `10_项目/`：项目上下文、决策、任务、问题和风险
- `20_资料库/`：原始资料和初步整理资料
- `30_原子笔记/`：长期可复用知识点
- `35_主动回忆/`：回忆卡片和复习队列
- `60_行业情报/`：外部变化信号
- `65_洞察/`：跨资料和项目形成的高价值判断
- `75_提示词库/`：可复用人工智能协作提示词
- `80_复盘/`：周、月、项目复盘
- `85_指标/`：输入、处理、吸收、洞察和转化指标
