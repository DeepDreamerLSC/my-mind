# 收件箱录入

## 用途

把用户随手丢来的链接或片段先安全写入 `00_收件箱/`，等待后续整理。

## 当前能力

- 支持一个或多个链接。
- 优先用 `yt-dlp` 抓取视频或网页基础信息。
- 记录标题、作者或频道、发布时间、时长、平台、简介摘录、标签、分类、缩略图和字幕可用性。
- 对 YouTube 链接，会尝试从公开视频页发现官方转录链接，并记录转录来源、章节目录、片段数和估算词数。
- 如果 `yt-dlp` 提供字幕轨道，会解析字幕来源、语言、格式、片段数和估算词数。
- 抓取失败时仍然生成收件箱笔记，并记录失败原因。

## 使用方式

```text
收件箱录入：
https://www.youtube.com/watch?v=...
```

Codex 应调用：

```bash
python3 .codex/skills/inbox-capture/scripts/capture_link.py "链接"
```

## 处理边界

- 只入箱，不晋升为长期知识。
- 不下载视频或音频。
- 不做 Whisper 转写。
- 不使用登录态或 cookie。
- 不把超长版权转录全文复制进仓库，只保留可追溯来源和结构化解析结果。
- 不默认复制字幕原文或歌词。
